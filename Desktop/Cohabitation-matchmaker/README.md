# Cohabitation matcher 

# Description
This is a web application designed to help individuals find compatible roommates based on preferences, lifestyle, and location.

# Objectives
- Help users find suitable cohabitants
- Reduce conflicts by matching compatible personalities
- Provide an easy-to-use platform for roommate search

# Features (Planned)
- User registration and login
- Profile creation
- Matching system
- Communication system
- Preferences filtering (budget, habits, location)

# Technology Stack
- Frontend: (e.g. / HTML / CSS / javascript )
- Backend: (e.g. Node.js )
- Database: (e.g. MySQL / MongoDB)
